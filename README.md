# Telecom Control Panel API

FastAPI-based API to interact with Kubernetes clusters using kubectl.

## Features
- Get Pods (/pods)
- Kubernetes integration via subprocess
- Swagger UI support

## Tech Stack
- FastAPI
- Kubernetes
- Python

## Setup
pip install fastapi uvicorn
uvicorn main:app --reload
